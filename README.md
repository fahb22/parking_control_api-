Estudando alguns conceitos com a Michelle Brito, exploro tanto a parte teórica quanto prática do Spring, Inversão de Controle, Injeção de Dependências, Beans, definição Spring Boot e também a construção de um projeto Spring Boot (Parking Control API) do zero, com projetos como Spring MVC, Spring Data JPA e Spring Validation para a criação de uma API com Pageable, métodos getAll, getOne, post, delete e update.
